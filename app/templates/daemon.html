Daemon Process

Proses latar belakang yang berjalan terus-menerus

01

Apa itu Daemon?

Daemon adalah proses komputer yang berjalan di latar belakang, bukan di bawah kendali langsung pengguna. Daemon biasanya dimulai saat boot dan berjalan terus hingga sistem dimatikan. Contoh: server web, database, scheduler.

$ systemctl status sshd
02

Membuat Daemon Sederhana

Berikut script daemon monitoring sistem dari tugas kelompok:

#!/bin/bash
WORK_DIR="/opt/uji_daemon"
LOG_FILE="$WORK_DIR/uji.log"
PID_FILE="/var/run/uji_daemon.pid"

monitor_system() {
    while true; do
        cpu=$(top -bn1 | grep "Cpu(s)" | awk '{print $2}')
        mem=$(free -m | awk 'NR==2{printf "%.2f%%", $3*100/$2}')
        disk=$(df -h / | awk 'NR==2{print $5}')
        echo "$(date) - CPU: $cpu% MEM: $mem DISK: $disk" >> $LOG_FILE
        sleep 300
    done
}
03

Systemd Service

Mendaftarkan daemon sebagai service systemd agar auto-start saat boot:

[Unit]
Description=System Monitor Daemon
After=network.target

[Service]
Type=forking
ExecStart=/opt/uji_daemon/uji_daemon.sh start
ExecStop=/opt/uji_daemon/uji_daemon.sh stop
PIDFile=/var/run/uji_daemon.pid
Restart=always

[Install]
WantedBy=multi-user.target
