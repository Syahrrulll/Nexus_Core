Panduan Docker

Container, Image, Compose & Deployment

01

Apa itu Docker?

Docker adalah platform untuk mengembangkan, mengirim, dan menjalankan aplikasi di dalam container. Container memungkinkan aplikasi dipaketkan dengan semua dependensinya, sehingga bisa berjalan konsisten di lingkungan manapun.

$ docker --version
02

Perintah Dasar Docker

docker pull <image>Tarik image dari registry
docker run -d -p 8080:80 <image>Jalankan container
docker ps -aLihat semua container
docker stop <container>Hentikan container
docker build -t <name> .Build image dari Dockerfile
docker exec -it <container> bashMasuk ke container
03

Docker Compose

Docker Compose digunakan untuk menjalankan multi-container application. Contoh dari tugas projek akhir:

version: '3.8'
services:
  web:
    build: .
    ports:
      - "8000:8000"
    depends_on:
      - mysql
  mysql:
    image: mysql:5.7
    environment:
      MYSQL_ROOT_PASSWORD: awvx0563
    volumes:
      - mysql_data:/var/lib/mysql
volumes:
  mysql_data:
04

Bedah Container

Teknik investigasi container untuk memahami isi, konfigurasi, dan proses di dalamnya:

docker inspectDetail container
docker execMasuk container
docker statsResource usage
docker historyLayer image
docker saveExport image
