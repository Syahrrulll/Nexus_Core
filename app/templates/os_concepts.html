01

Apa itu Arsitektur Sistem?

Sistem operasi adalah perangkat lunak inti pada komputer yang mengelola perangkat keras dan menyediakan layanan untuk menjalankan aplikasi. OS bertindak sebagai penghubung antara hardware dan software, memungkinkan program dijalankan dengan efisien.

02

Fungsi Utama OS

Manajemen Proses — Mengatur eksekusi program
Manajemen Memori — Mengalokasi RAM ke program
Manajemen Storage — Mengatur penyimpanan data
Manajemen I/O — Mengontrol perangkat input/output
Keamanan — Melindungi data dan resource
03

Komponen Arsitektur Sistem

User Applications Browser, Office, Games
Shell / GUI User Interface
System Calls API Interface
Kernel Core OS
Hardware CPU, RAM, Disk
